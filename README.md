# 智驾测试记录仪

这是一个手机浏览器可运行的辅助驾驶数据记录 App，用于记录加速度、速度、GPS 位置，并识别急刹车、急加速、急转向和人工标记事件。

当前手机端负责采集和导出原始路测数据；后续电脑端可以作为独立项目读取 JSON/CSV，把数据渲染成带透明通道的赛车 HUD 视频，再叠加到实车测试视频上。

## 本地运行

```bash
npm start
```

打开 `http://localhost:4173`。手机实车使用时建议部署到 HTTPS 地址；iOS 的运动传感器授权必须由按钮触发。

桌面预览时可点「演示」生成模拟速度、加速度、轨迹和驾驶事件，用来检查可视化与导出流程。

实车上建议把本目录作为静态站点部署到 HTTPS 环境后用手机访问。若只在电脑本机预览，`localhost` 可以打开页面；若让手机访问电脑局域网地址，多数浏览器会因为不是 HTTPS 而限制运动传感器或定位权限。

## 传感器矫正

- 静止校准：车辆静止、手机固定后采集重力方向、线性加速度偏置和噪声。
- 手机朝向：在无法做直线校准时，用手机顶端、右侧、底端或左侧朝车头来建立车辆前向轴。
- 直线校准：直线行驶中利用 GPS 速度变化和手机加速度方向，自动修正手机相对车辆的前后方向。
- 校准角度：静止校准后显示手机俯仰、横滚；直线校准后显示前向偏角，便于确认安装方向是否可信。

## 定位与速度

- 定位卡片会显示 GPS 是否锁定，以及实时经纬度坐标。
- 速度优先使用手机系统提供的 GPS speed；当系统没有提供 speed 时，才用连续 GPS 坐标差分估算，并通过定位精度和跳变过滤降低静止漂移造成的误差。
- 导出数据保留 8 位小数经纬度、定位精度、海拔、航向、速度来源和 GPS 原始速度字段。

## 导出数据

记录结束后可导出 CSV 或 JSON。CSV 包含时间、GPS 速度、速度来源、高精坐标、定位精度、原始三轴加速度和车辆坐标系下的纵向/横向/垂向加速度。JSON 使用 `adas-recorder-session-v2` schema，适合作为电脑端 HUD 视频生成项目的数据输入。
